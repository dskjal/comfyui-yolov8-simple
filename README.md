Simple yolov8 comfyui plugin

## INSTALL

1. in comfyui `custom_nodes` dir and `https://github.com/dskjal/comfyui-yolov8-simple.git`
2. put detect or seg models in comfyui `models/yolov8` dir

## How to use

If "seg" or "Seg" or "SEG" in your yolov8 model name, the node outputs segmentation mask.

Node name is Yolov8DetectionSegmentation.
